# front-end-andrea

# Meu Projeto Front-End

Este é um projeto desenvolvido com [Vite](https://vite.dev/guide/) e React, como parte da atividade da disciplina de Front-End.

##  Descrição da aplicação
Aplicação simples que exibe um componente com propriedades. O tema será expandido nas próximas aulas.

##  Tecnologias
- Vite
- React
- JavaScript

##  Como executar
1. Clone este repositório
2. Instale as dependências com `npm install`
3. Execute com `npm run dev`
